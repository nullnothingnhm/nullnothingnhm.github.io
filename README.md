## 🛠️ Customization

### Update Contact Information
Edit the contact section in `index.html`:
```html
<a href="mailto:your.email@example.com" class="contact-card">
    <i class="fas fa-envelope"></i>
    <h3>Email</h3>
    <p>Get in touch</p>
</a>
```

### Change Social Links
Update the social links in the contact section:
- GitHub: `https://github.com/nullnothingnhm`
- LinkedIn: `https://linkedin.com/in/nullnothingnhm`
- Email: Your email address

### Modify Content
All content is in the `index.html` file. Simply edit the text, dates, and descriptions to match your updated information.

### Customize Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --accent-primary: #00ff88;  /* Neon Green */
    --accent-secondary: #00d9ff;  /* Cyan */
    --accent-danger: #ff0055;  /* Red */
}
```

## 🚀 Deployment

Your site is automatically deployed via GitHub Pages. To make changes:

1. Edit `index.html`, `styles.css`, or `script.js`
2. Commit your changes: `git commit -m "Update portfolio"`
3. Push to GitHub: `git push origin main`
4. Your site updates automatically in a few seconds!
